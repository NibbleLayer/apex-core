# @nibblelayer/apex-contracts

Public Apex contracts package.

## Purpose

This package exposes the transport-facing schemas and TypeScript types used by Apex clients, SDKs, and control-plane integrations.

## Install

```bash
pnpm add @nibblelayer/apex-contracts
```

## Exports

- `@nibblelayer/apex-contracts`
- `@nibblelayer/apex-contracts/schemas`

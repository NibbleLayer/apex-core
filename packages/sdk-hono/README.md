# @nibblelayer/apex-hono

Public Apex Hono SDK.

## Purpose

This package integrates a Hono application with an Apex control plane so protected routes can resolve manifest configuration and apply x402 payment middleware.

## Install

```bash
pnpm add @nibblelayer/apex-hono hono
```

## Example

```ts
import { createApexClient } from '@nibblelayer/apex-hono';
```

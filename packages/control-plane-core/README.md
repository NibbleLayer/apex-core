# @nibblelayer/apex-control-plane-core

Public Apex control-plane kernel helpers.

## Purpose

This package provides manifest construction utilities that sit between persisted Apex configuration and seller-facing runtime consumers.

## Install

```bash
pnpm add @nibblelayer/apex-control-plane-core
```

## Example

```ts
import { buildManifest } from '@nibblelayer/apex-control-plane-core';
```
